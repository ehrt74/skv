* skv

simple key value store.

a minimal key value store for bash scripting

** Usage

#+begin_src bash
  $ skv put my-scheme mykey foobar
  $ skv get my-scheme mykey
  foobar
  $ skv scheme-ls
  my-scheme
#+end_src


** Installation

1. install sbcl and ocicl
2. run ~make build~
3. mv ~skv~ to your $PATH
4. load ~skv-completions.bash~
